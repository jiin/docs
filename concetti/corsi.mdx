Nel sistema, un **corso** è una voce del catalogo UTRtek: ad esempio "Open Water Diver", "Advanced Open Water", ecc.

## Cosa contiene la scheda di un corso

| Informazione | A cosa serve |
|--------------|--------------|
| **Nome** | Il nome ufficiale del corso |
| **Codice SKU** | Codice interno per identificarlo |
| **Prezzo** | Costo della PIK per quel corso |
| **Profondità massima** | Limite di profondità previsto |
| **Durata** | Ore o giorni del corso |
| **Prerequisiti per l'allievo** | Cosa deve aver già fatto l'allievo |
| **Prerequisiti per insegnare** | Cosa deve avere l'istruttore |
| **Titoli al completamento** | Quali gradi/badge si ottengono |
| **Rinnovabile** | Se la PIK si può rinnovare anno dopo anno |
| **Collegamento Moodle** | Link alla piattaforma didattica |

## I prerequisiti

Un corso può richiedere che l'allievo abbia già:

- Completato un **altro corso UTRtek** (es. serve Open Water prima di Advanced)
- Ottenuto un certo **titolo** (grado subacqueo)
- Dichiarato una **licenza personale** (brevetto di un'altra agenzia)

Il sistema **controlla automaticamente** se l'allievo soddisfa i prerequisiti quando si emette una PIK.

## Il diagramma corsi

Gli amministratori possono aprire una **mappa visiva** (diagramma) che mostra come i corsi sono collegati tra loro tramite i prerequisiti.

Per aprirla: vai in **Didattica e PIK → Corsi** e cerca il pulsante **Diagramma corsi**.

## Corsi rinnovabili

Alcuni corsi permettono il **rinnovo annuale** della PIK. Questo significa che, alla scadenza, si può creare una nuova PIK con validità prolungata senza rifare tutto il corso.
