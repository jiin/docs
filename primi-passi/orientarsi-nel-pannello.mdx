Dopo il login vedi il **pannello UTRtek**. Ecco le parti principali dello schermo, spiegate una per una.

## Le parti dello schermo

```
┌─────────────────────────────────────────────────────────┐
│  [Logo UTRtek]     🔍 Cerca...          🔔  👤 Nome    │  ← Barra in alto
├──────────────┬──────────────────────────────────────────┤
│              │                                          │
│  MENU        │         AREA PRINCIPALE                  │
│  laterale    │         (contenuto della pagina)         │
│              │                                          │
│  Didattica   │                                          │
│  Documenti   │                                          │
│  Anagrafica  │                                          │
│  Sistema     │                                          │
│              │                                          │
└──────────────┴──────────────────────────────────────────┘
```

### 1. Barra in alto

- **Logo UTRtek** — clic per tornare alla Dashboard
- **Cerca** — barra di ricerca globale (scorciatoia: `Ctrl+K` su Windows o `Cmd+K` su Mac)
- **Campanella** — notifiche del sistema
- **Il tuo nome** — menu utente (logout, lingua, ecc.)

### 2. Menu laterale (a sinistra)

È diviso in **4 gruppi** (dall'alto verso il basso):

| Gruppo | Contenuto |
|--------|-----------|
| **Didattica e PIK** | Corsi, PIK Cards, Rinnovi, Licenze personali |
| **Documenti e strutture** | Certificato medico, assicurazione, la mia struttura |
| **Anagrafica e gestione** | Utenti, strutture, ruoli, titoli, documenti |
| **Sistema** | Template email, storico invii |

Le voci che vedi dipendono dal **tuo ruolo**. Un allievo vede meno voci di un amministratore.

### 3. Area principale

Qui compare il contenuto della pagina che hai scelto: tabelle, moduli, grafici, ecc.

## La Dashboard (pagina iniziale)

È la prima cosa che vedi dopo il login. Contiene **riquadri informativi** (widget) diversi a seconda del ruolo:

**Se sei amministratore:**
- Statistiche generali (acquisti, rinnovi, nuovi utenti)
- Grafici dell'andamento
- Tabelle con PIK recenti, corsi più frequenti, migliori istruttori

**Se sei allievo o istruttore (non admin):**
- **Il tuo riepilogo:** stato certificato medico, assicurazione, numero PIK, ruolo
- Eventuali avvisi in rosso se mancano documenti

## La ricerca globale

In alto c'è una **lente di ingrandimento** o un campo "Cerca".

<Steps>
  <Step title="Apri la ricerca">
    Clicca sulla barra di ricerca, oppure premi `Ctrl+K` (Windows) o `Cmd+K` (Mac).
  </Step>
  <Step title="Digita cosa cerchi">
    Puoi cercare: utenti, PIK Card, corsi, strutture formative, titoli.
  </Step>
  <Step title="Seleziona il risultato">
    Clicca sul risultato per aprire direttamente quella scheda.
  </Step>
</Steps>

## Le tabelle (elenchi)

Molte pagine mostrano una **tabella** con righe di dati. Le tabelle hanno sempre:

- **Filtri** — per restringere i risultati (es. solo PIK attive)
- **Colonne ordinabili** — clic sull'intestazione per ordinare
- **Azioni per riga** — icone o pulsanti a destra di ogni riga (visualizza, modifica, ecc.)
- **Pulsante Crea** — in alto a destra, per aggiungere un nuovo elemento

## I pulsanti comuni

| Pulsante | Cosa fa |
|----------|---------|
| **Crea** / **Nuovo** | Aggiunge un nuovo record |
| **Salva** | Conferma le modifiche |
| **Annulla** | Torna indietro senza salvare |
| **Visualizza** (occhio) | Apre la scheda in sola lettura |
| **Modifica** (matita) | Apre il modulo di modifica |
| **Elimina** (cestino) | Rimuove il record (chiede conferma) |
